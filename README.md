# VNR Academic Repository

A web platform for students to upload, preview, and download academic materials.

## Features

- Google Authentication
- Admin moderation system
- File preview and download
- Folder-based subject organization
- Student upload system

## Tech Stack

Frontend:
React + Vite

Backend:
Node.js + Express

Database:
MongoDB Atlas

Authentication:
Firebase

Deployment:
Vercel + Render
